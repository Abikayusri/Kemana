== User
Kemana User API (Driver and Passenger)

// save user //
=== Save User
Save user to document

===== Sample Request
include::{snippets}/user/save-user/http-request.adoc[]

|===
| param | desc

| `document`
| Document where user saving, `driver`, `passenger` or `driver_active`

|===

====== Request Body
For request body is object user, all nullable
[source]
----
{
  "name" : "marjuki",
  "email" : "email@marjuki.com",
  "photo" : "photoUrl"
}
----

====== User Object
[source]
----
{
    "id": "1281d0ac-7a74-3b91-950f-f52a02862cda",
    "name": "marjuki",
    "email": "email@marjuki.com",
    "phone": "0812983029",
    "photo": "http://example.jpg",
    "position": {
      "lat": -6.2242497,
      "lon": 106.9240669
    },
    "driver_attribute": {
      "num": "B 8773 A",
      "type": "HONDA",
      "color": "white"
    }
}
----
